# flutter_postal

这是一个练习项目，只实现了 UI 设计，不包含业务逻辑。

在 IPhone X 下开发，其他屏幕未做适配，页面显示可能有问题。

UI 设计来自 [http://freebielobster.com/index.php/home/post/Postal-App-Ui-Freebie)，仅供学习使用。
